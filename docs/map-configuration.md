## 🚀 使用方法

### 从行程详情页访问地图
1. 打开任意行程详情页面
2. 点击操作栏中的 "🗺️ 地图" 按钮
3. 系统将跳转到地图页面，显示该行程的所有地点

### 地图页面操作
- **定位按钮**: 点击右下角的 "📍" 定位按钮回到当前位置
- **路线切换**: 点击右上角的 "🗺️" 按钮切换路线显示
- **标记点击**: 点击地图上的标记查看详细信息
- **地图点击**: 在地图空白处点击可添加自定义标记

### 天气信息
- 左上角显示当前位置的天气信息
- 包括温度、天气状况和城市名称
- 自动根据位置更新天气数据

## 🔧 技术实现

### API集成
- **腾讯地图API**: 提供地图基础服务和地理编码
- **华为云市场天气API**: 提供实时天气信息
- **微信小程序Location API**: 获取用户位置权限

### 数据流
1. 用户从行程详情页点击地图按钮
2. 地图页面接收行程ID和目的地参数
3. 调用腾讯地图API进行地理编码
4. 在地图上显示标记点和路线
5. 获取并显示当前天气信息

### 错误处理
- API调用失败时使用默认坐标
- 网络异常时显示友好提示
- 权限被拒绝时提供替代方案
