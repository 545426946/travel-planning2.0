## 技术实现细节

### 景点提取算法
```javascript
// 支持多种模式识别景点
const attractionPatterns = [
  /([^，。；\n]{2,20})(?:[（(]地址[：:]\s*([^）)]{5,30})[）)])/g,  // 带地址
  /(?:在|前往|参观|游览|去到|到达|探访)([^，。；\n]{2,20})/g,     // 带动作
  /([^，。；\n]*(?:故宫|长城|颐和园|...)[^，。；\n]*)/g,           // 关键词
  /[🌅☀️🌙]\s*(?:上午|下午|晚上)[^：:]*[：:]\s*([^，。；\n]{2,30})/g, // 时间段
  /(参观|游览|前往|去到|探访|寻访)([^，。；\n]{2,20})/g,           // 动作+景点
  /([^，。；\n]{2,15}(?:公园|博物馆|寺|庙|塔|阁|宫|殿|观|湖|海|湾|岛|街|巷|场|广场|古镇|遗址|景区|景点))/g
]
```
